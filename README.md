### Diccionario de Datos del DATASASET

A continuación, se presenta un diccionario de datos inicial para las columnas del DATASET:

| Columna             | Tipo de Dato | Descripción                                                                    |
|:--------------------|:-------------|:-------------------------------------------------------------------------------|
| `Name`              | `object`     | Nombre completo del paciente.                                                  |
| `Age`               | `int64`      | Edad del paciente en años.                                                     |
| `Gender`            | `object`     | Género del paciente (e.g., 'Male', 'Female').                                  |
| `Blood Type`        | `object`     | Tipo de sangre del paciente (e.g., 'A+', 'B-').                                |
| `Medical Condition` | `object`     | Condición médica diagnosticada del paciente (e.g., 'Cancer', 'Obesity').       |
| `Date of Admission` | `object`     | Fecha de admisión del paciente al hospital. (Actualmente como cadena de texto) |
| `Doctor`            | `object`     | Nombre del médico que atiende al paciente.                                     |
| `Hospital`          | `object`     | Nombre del hospital.                                                           |
| `Insurance Provider`| `object`     | Proveedor de seguros del paciente.                                             |
| `Billing Amount`    | `float64`    | Cantidad facturada al paciente.                                                |
| `Room Number`       | `int64`      | Número de habitación asignada al paciente.                                     |
| `Admission Type`    | `object`     | Tipo de admisión (e.g., 'Elective', 'Urgent').                                 |
| `Discharge Date`    | `object`     | Fecha de alta del paciente del hospital. (Actualmente como cadena de texto)    |
| `Medication`        | `object`     | Medicación recetada al paciente.                                               |
| `Test Results`      | `object`     | Resultados de las pruebas médicas (e.g., 'Abnormal', 'Normal').                |
